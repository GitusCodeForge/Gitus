Gitus Web Installer

Welcome to the Gitus Web Installer. We'll guide you through the installation process.

In the end this installer would set up the database and produce a configuration file. At that point, please run Gitus using that configuration file like this:

gitus -config {config-file-path}

NOTE THAT the web installer will skip some steps if they're not compatible or don't make sense with certain decisions you have made during the earlier part in the installation process (e.g. asking for database info when you've decided to have a setup that shouldn't need a database). This is normal and by design.
